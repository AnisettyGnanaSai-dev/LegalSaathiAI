# LegalSathi AI

**LegalSathi AI** is a production-oriented Retrieval-Augmented Generation (RAG) workflow copilot designed to assist citizens in understanding consumer rights and the Right to Information (RTI) Act.

## Architecture
- **LLM:** Qwen 2.5 7B
- **Embeddings:** BAAI/bge-m3
- **Vector DB:** Qdrant
- **Framework:** LangChain & FastAPI

*Note: Data files and vector stores are excluded from this repository. They must be uploaded directly to your compute environment.*
